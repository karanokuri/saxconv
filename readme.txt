 はじめに

サックスで演奏（録音）したオーディオデータを移調させるソフトです。
ぼっちでも、サックスが一本しかなくても、サックスアンサンブルが実現します。

-------------------------------------------------------------------------------
 使い方

サックスを演奏します。
楽譜を読み換える必要はありません。楽譜の通りお吹きください。

左側から移調する前の（演奏に用いた）サックスを選びます。
右側で移調させたいサックスを選びます。
変換ボタンを押すと移調後のオーディオファイルとして書き出されます。
